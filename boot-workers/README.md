## Workers

You can add this module to your project by inserting the following dependency to **pom.xml**:
```xml
<dependency>
    <groupId>io.datakernel</groupId>
    <artifactId>datakernel-workers</artifactId>
    <version>3.1-SNAPSHOT</version>
</dependency>
```
